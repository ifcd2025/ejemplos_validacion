Validaciones con HTML y JavaScript

Los requisitos se deben establecer en HTML pero, al pulsar el botón Enviar, se deben crear un alert con todos los problemas. Debe estar desactivada la validación en el formulario para que no salgan los mensajes de error del navegador

Hay datos incorrectos

  • El nombre, edad y la clave son requeridos
  • El nombre debe tener 3 caracteres como mínimo
  • La edad debe estar entre 18 y 100
  • La clave debe tener 6 caracteres como mínimo
