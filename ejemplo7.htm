Validaciones con HTML y JavaScript

Los requisitos se deben establecer en HTML pero, al pulsar el botón Enviar, se deben mostrar todos los problemas en el ul errores. También debe ponerse de color rojo con JavaScript los elementos erróneos. Debe estar desactivada la validación en el formulario para que no salgan los mensajes de error del navegador. Además, dejar escribir solo números en la edad

    • El nombre, edad y la clave son requeridos
    • El nombre debe tener 3 caracteres como mínimo
    • La edad debe estar entre 18 y 100
    • La clave debe tener 6 caracteres como mínimo
    • Debe seleccionarse la provincia
